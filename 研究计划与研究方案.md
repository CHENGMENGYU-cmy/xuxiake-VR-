## 2.4 核心模块方案

### 用户认证方案
- 方式：邮箱注册 + JWT认证
- Token：Access Token（2小时） + Refresh Token（7天）
- 存储：Redis存Session，MySQL存用户信息

### 实时通信方案
- 技术：WebSocket（Spring Boot + STOMP）
- 场景：私信聊天、通知推送、在线状态
- 消息存储：MySQL持久化 + Redis暂存未读

### 推荐算法方案
- 内容推荐：协同过滤（用户行为） + 内容特征（标签/地点） + 热度衰减
- 社交匹配：多维特征向量相似度（兴趣/年龄/地区/活跃度）
- 冷启动：基于注册时选择的兴趣标签 + 热门内容兜底

### AI游记生成方案
- 输入：AR轨迹数据 + 语音转写 + 拍摄照片
- 处理：大模型API，Prompt模板化
- 输出：图文并茂的游记草稿，用户可编辑发布
